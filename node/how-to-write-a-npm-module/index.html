    #注册登录

    • 如没有账号需到npm账号注册一个
    • 如已注册成功，在shell里输入npm login，按照提示输入对应的用户名、密码等即可登录

    #初始化

    npm init -y

    #创建npm包

    exports.sayHello = function() {
  console.log("hello world");
}

    #发布

    npm publish
